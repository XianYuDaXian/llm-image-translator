# llm-image-translator 隐私政策（草案）

最后更新：2026-04-28

`llm-image-translator` 是一个浏览器扩展，用于将网页中的图片发送到用户自行配置的大模型图像接口，并返回翻译后的结果图。

## 1. 扩展会处理哪些数据

扩展可能会处理以下数据：

- 用户当前页面中的图片内容
- 图片 URL 与当前页面 URL
- 用户在设置页填写的模型服务配置
  - `Base URL`
  - `API Key`
  - `Model`
  - 自定义请求参数
- 扩展生成的图片缓存与任务状态

## 2. 数据如何使用

这些数据仅用于以下目的：

- 识别当前网页中的可翻译图片
- 将用户选中的图片发送到用户配置的模型服务进行翻译
- 在页面中展示翻译结果
- 缓存翻译结果，减少重复请求

## 3. 数据会发送给谁

扩展不会默认把数据发送到开发者自有服务器。

当用户启用翻译时，图片及相关请求数据会发送到**用户自行配置的第三方模型服务端点**。这些端点可能是：

- OpenAI 官方接口
- Gemini 官方接口
- xAI 兼容接口
- OpenAI / Gemini 兼容接口
- 用户自定义的私有服务或局域网服务

用户应自行确认目标服务的可信度、可用性和隐私政策。

## 4. 本地存储

扩展会在浏览器本地存储以下信息：

- 用户的扩展设置
- 最近任务状态
- 排除翻译的网站与图片
- 翻译结果缓存
- 对于 `base64` / 二进制结果图，扩展会将图片二进制缓存到浏览器本地 `IndexedDB`

这些数据用于扩展功能本身，不会主动同步到开发者服务器。

## 5. 数据共享与出售

开发者不会出售用户数据，也不会将扩展收集的数据主动共享给无关第三方。

## 6. 用户控制

用户可以：

- 修改或删除模型服务配置
- 清除浏览器中的扩展存储数据
- 停用或卸载扩展
- 配置排除翻译的网站
