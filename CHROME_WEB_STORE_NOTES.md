# Chrome Web Store 提审说明（草案）

## 单一用途

`llm-image-translator` 的唯一用途是在网页中识别图片，并调用用户自行配置的大模型图像接口生成翻译后的结果图。

## 为什么需要 `<all_urls>`

该扩展的核心交互是：

- 在任意网页图片上显示悬浮翻译按钮
- 支持右键翻译当前图片
- 支持当前页面图片任务面板

因此扩展需要在用户访问的网页中注入内容脚本，以识别页面中的 `<img>` 元素并在图片上方提供交互入口。

## 为什么支持自定义端点

该扩展是一个模型接口客户端，不内置单一服务商。用户可以根据自己的使用场景配置：

- 官方 API
- 兼容 API
- 私有部署接口
- 局域网接口

扩展不会默认把数据发送到开发者服务器，请求只会发往用户显式配置的模型端点。

## 建议写入商店简介的说明

- 本扩展会将用户选择翻译的网页图片发送到你在设置页中配置的模型服务端点。
- 请仅在你信任的模型服务上使用本扩展。
- 如果你配置的是自托管或局域网接口，请自行确认网络与数据安全。
